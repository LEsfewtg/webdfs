# ExamShield Dashboard

## Chạy local

```bash
npm install
npm run dev
```

## Deploy Vercel

- Framework Preset: Vite
- Root Directory: thư mục chứa package.json
- Build Command: npm run build
- Output Directory: dist
- Install Command: để mặc định

Bản này không kèm package-lock.json để tránh khóa URL registry nội bộ không dùng được trên Vercel.
