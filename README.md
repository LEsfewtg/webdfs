# ExamShield Dashboard

Trang web mô phỏng giao diện giám sát phòng học, xây bằng React + Vite và có thể deploy trực tiếp lên Vercel.

## Chạy trên máy

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy lên Vercel

1. Đưa toàn bộ thư mục lên GitHub.
2. Vào Vercel → Add New Project.
3. Chọn repository vừa tạo.
4. Framework Preset: Vite.
5. Build Command: `npm run build`.
6. Output Directory: `dist`.
7. Deploy.

Dự án là bản mô phỏng front-end, dữ liệu được tạo sẵn và tự thay đổi nhẹ để tạo cảm giác dữ liệu trực tuyến.
